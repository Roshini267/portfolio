# Roshini Thalari - Personal Portfolio

Personal portfolio website showcasing engineering qualifications, project distributions, and professional technical internships. Built as a fully responsive single-page web platform using custom HTML5 structures, CSS3 design systems, and modern interactive vanilla JavaScript behaviors.

## Project Overview

This website serves as a live developer profile detailing core competencies, professional milestones, and technical layouts:
*   **About Section**: Overview of academic pathways and core objectives in technical engineering spaces.
*   **Resume Section**: Complete timeline mapping out academic credentials, framework certifications, and multiple engineering internships.
*   **Portfolio Section**: Display portal for personal projects including responsive layout designs and UI testbeds.
*   **Contact Interface**: Modular interactive configuration featuring email links, active phone parameters, and message input fields.

## Prerequisites

Before setting up or deploying this project locally, ensure you have met the following platform requirements:

*   [Git](https://git-scm.com/downloads "Download Git") version control management client installed on your computer.
*   [Visual Studio Code](https://code.visualstudio.com/) text editor or a similar integrated development environment.
*   **Live Server** extension installed in VS Code to view responsive styles locally.
